Search Delivery Reference Number in SO

The default Odoo functionality provided a unique reference number for each delivery created. We developed this module to add an extra spark to this functionality of Odoo, i.e. with the help of delivery number you can now search that delivery and get its information easily.

You can search through delivery number:

  • In group by
  • Direct search
  • Through filters.

You can also get the already existing sale orders filtered through the above methods. You just need to configure it.

Prerequisites before Installation

No prerequisites needed.

Configurations

No configurations.

As soon as you confirm the sale order a delivery number is created for that particular sale order.

Under the “Other Information” tab you can view particular sale order’s delivery number.

As shown in the below image once you get the delivery number generated for the sale order, you can search through the delivery number itself from the search bar.

You can also search the sale order through the delivery number in Group By.

We have added the “Delivery Number” filter within the filter options. So, you can also filter the sale orders by delivery number filter.

For the situations, when you also want to get the sale orders created before the installation of this module to be filtered with the delivery number, you need to configure it by following steps:

Go to> Configurations> Click on “Delivery Number Existed So”.

Once you have configured the above-mentioned requirement, a pop up will be displayed on your screen for confirmation. Click confirm to get the already existing sale orders to be filtered by delivery number.
